¿Para quién está diseñado?

El programa está dirigido a todas las personas con responsabilidades o interés relacionado con el desarrollo de procesos de análisis de datos y desarrollo de sistemas inteligentes y analíticos

Personas que se desarrollen cómo: gerentes de informática, analistas, desarrolladores. El programa también está dirigido a profesionales en áreas cómo: Ingeniería, ciencias naturales, economía, matemática y estadística, interesados en el aprendizaje computacional.

Se requiere que los estudiantes tengan familiaridad previa con conceptos de programación; sin embargo, si no poseen estos conceptos, pueden iniciar con el curso gratuito preparatorio, creado especialmente para iniciar este programa.

¿Qué vas a aprender?

En este programa certificado de formación online te aproximarás de forma práctica y profunda a las herramientas más utilizadas y demandadas para el desempeño efectivo en las áreas de Machine Learning & Data Science.

Planeación y ejecución de proyectos de análisis de datos

Hallar características relevantes y relaciones importantes entre datos

Resolver problemas de análisis de datos mediante técnicas de aprendizaje computacional

Implementar soluciones mediante librerías especializadas en el lenguaje Python

Desarrollar soluciones computacionales con almacenamiento, procesamiento y acceso a grandes volúmenes de datos

Ejecutarás proyectos en los que se requieran tecnologías Big Data
